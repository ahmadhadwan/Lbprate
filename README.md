Lbprate
=======

This tool fetches the current 1 USD to LBP buy and sell rate
from https://lbprate.com/

# Dependencies
- libcurl

# Build Dependencies
- gcc
- make

# Building
```
    git clone https://github.com/ahmadhadwan/Lbprate
    cd Lbprate
    make
```

# Usage
```
    ./bin/lbprate --help
```
